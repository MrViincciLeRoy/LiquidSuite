About LSuite

Ledger Suite - Financial Integration Platform

LSuite is a modern Flask-based financial management system that seamlessly integrates Gmail bank statements with ERPNext accounting.

Key Features

  • Gmail Integration - OAuth 2.0 secure authentication
  • PDF Parsing - Extract transactions from bank PDFs
  • Auto-Categorization - Keyword-based matching
  • ERPNext Sync - Seamless journal entry creation
  • Multi-Bank Support - TymeBank, Capitec, and more
  • Bulk Operations - Process multiple transactions
  • Transaction Tracking - Monitor sync status
  • RESTful API - Programmatic access

Technology Stack

  • Backend: Flask, SQLAlchemy, PostgreSQL
  • Frontend: Bootstrap 5, Font Awesome
  • APIs: Google Gmail API, ERPNext API
  • Background Tasks: Celery, Redis
